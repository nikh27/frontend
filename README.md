# Local Development Steps (in frontend folder)

1. Install Node v8.19.2

2. Install Yarn

```shell
npm i -g yarn
```

3. Install Dependencies

```shell
yarn install
```

4. Start Development Server

```shell
yarn run dev
```

5. Production Build (dist) folder

```shell
yarn run build
```
